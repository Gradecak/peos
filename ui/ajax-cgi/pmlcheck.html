PMLcheck Submit Form

This is a very minimal pmlcheck interface designed to demonstrate the round-trip process for submitting pml code to a server-side cgi program. It simply sends the contents of the editor pane to the cgi program, then displays the output.

Ideally, you would want to submit the form via jquery's ajax() method so the pmlcheck output can be captured and inserted into an field next to the editor pane so the ouput can be compared to the input.
