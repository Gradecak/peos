Description

The action's script field would be displayed here. This should be formatted as an html paragraph (<p>), rather than text; we allow some html markup in the script field. Depending on the action state, buttons should be displayed:

  1. Active - display both ``Done'' (to finish the action), and ``OK'' (to dismiss the page and redisplay the worklist).
  2. Suspended - display both ``Perform'' (to make the action active), and ``OK'' (to dismiss the page and redisplay the worklist).
  3. Ready - display both ``Perform'' (so set the action to active) and ``OK'' (to dismiss the page and redisplay the worklist).
